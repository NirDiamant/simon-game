# Simon Game

A browser-based Simon memory game with neon visuals, synthesized audio, and polished gameplay. No frameworks, no dependencies — just HTML, CSS, and JavaScript.

**[Play it here](https://nirga.github.io/simon-game/)**

## Features

- Classic 4-pad Simon gameplay with progressive difficulty
- Web Audio API synthesized tones, SFX, and ambient drone
- Neon glow effects, particle system, and smooth animations
- Strict mode (one mistake = game over) and normal mode (replay on mistake)
- Score multipliers, high score persistence via localStorage
- Keyboard support (1/2/3/4 or Q/W/A/S)
- Fully responsive — works on desktop, tablet, and mobile

## How to Play

Open `index.html` in any modern browser. Watch the sequence, repeat it back. Each round adds one more step. How far can you go?
